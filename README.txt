Тестовое задание IPG.Estate
В качестве тестового задания просим сделать форму записи пациента на приём
html: дата/input, время/select (диапазоны), фио врача/select
jquery: submit формы через ajax запрос
php: обработка запроса, запрос в БД (mySQL), проверка занято время или нет у данного врача, ответ
Mysql: таблица(ы) на ваше усмотрение
в случае успеха - отправка на почту письма о записи.
